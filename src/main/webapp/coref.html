Co-reference can be used to find generic references to a person or object in the same sentence or across multiple sentences.
It may find Nick and determine in later text that the word he refers back to Nick.
When used in conjunction with an Entity Name Finder, it can locate names and determine that following personal pronoun in the text refers back to the original name.
This can be a difficult task and will not work 100% of the time.
